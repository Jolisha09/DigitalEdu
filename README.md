# DigitalEdu

DigitalEdu is a full-stack online course registration platform developed using Node.js, Express.js, SQLite, HTML, CSS, and JavaScript.

## Features

- Course Catalog
- Student Registration
- Responsive Design
- SQLite Database
- Full Stack Architecture

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- SQLite
- GitHub
- Render
